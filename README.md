# 🛡️ Hate Speech Detection using Machine Learning

An intelligent Machine Learning application that detects whether a given text contains **hate speech**, **offensive language**, or **neutral content**. This project leverages Natural Language Processing (NLP) techniques to preprocess text and train a classification model for automated moderation.

---

## 📌 Overview

Social media platforms generate millions of posts every day, making manual moderation difficult. This project aims to automatically classify user-generated text into predefined categories, helping create safer online communities.

---

## ✨ Features

- 🔍 Detects hate speech from input text
- 🧹 Text preprocessing and cleaning
- 🧠 Machine Learning-based text classification
- 📊 Model evaluation with accuracy and performance metrics
- 💻 Easy-to-use interface (if applicable)

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/imagam18/hate-speech-detection.git
cd hate-speech-detection
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

If using Jupyter Notebook:
